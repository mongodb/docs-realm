:::note Beta Release
This SDK is currently offered as a **beta** release. We encourage you to
try out the feature and [give feedback](https://feedback.mongodb.com/forums/923521-realm/).
However, be aware that APIs and functionality are subject to change.
:::

With the [Kotlin SDK](https://github.com/realm/realm-kotlin), you can
use Realm Database and Sync to build:

- Android and iOS applications written with [Kotlin Multiplatform Mobile
  (KMM)](https://kotlinlang.org/docs/mobile/home.html)
- Standalone Android applications

:::note Sync Support In Development
The SDK currently supports some, but not all, Realm Sync functionality.
For more information about supported behavior, see the
[API reference](https://www.mongodb.com/docs/realm-sdks/kotlin/latest/library-sync/io.realm.mongodb/index.html).
:::

## Local Realm Database

With the MongoDB Realm Kotlin SDK, you can access objects stored in a
local instance of Realm Database. With Realm Database, you can:

### Define an Object Schema

Define your object schema with marked Kotlin classes:

<CodeBlock language="kotlin">{Model}</CodeBlock>

### Query Realm Database

Query for stored objects:

<CodeBlock language="kotlin">{Query}</CodeBlock>

### Update Objects

Update objects in Realm Database by updating field values on an instance of the object within a transaction:

<CodeBlock language="kotlin">{Update}</CodeBlock>

## Get Started

To start using the MongoDB Realm Kotlin SDK in your application, see
the installation guide for [Kotlin Multiplatform](/sdk/kotlin/install/kotlin-multiplatform)
or [Android](/sdk/kotlin/install/android).

Once you've installed the SDK, check out the [Quick Start](/sdk/kotlin/quick-start).
