To find an object with a specific primary key value, open a realm
and query the primary key field for the desired primary key value
using [`realm.query()`](https://docs.mongodb.com/realm-sdks/kotlin/latest/library-base/-realm%20-kotlin%20-multiplatform%20-s-d-k/io.realm.query/-realm-query/query.html).
Specify the object type as a type parameter passed to `query()`:

<CodeBlock language="kotlin">{Find}</CodeBlock>

:::note `find()` is Synchronous

[`find()`](https://docs.mongodb.com/realm-sdks/kotlin/latest/library-base/-realm%20-kotlin%20-multiplatform%20-s-d-k/io.realm.query/find.html)
runs a synchronous query on the thread it is called from.
As a result, avoid using `find()` on the UI thread or in
logic that could delay the UI thread. Prefer [`asFlow()`](https://docs.mongodb.com/realm-sdks/kotlin/latest/library-base/-realm%20-kotlin%20-multiplatform%20-s-d-k/io.realm.query/-realm-element-query/as-flow.html)
in time sensitive environments.
:::
