It is possible to delete all objects from a realm. This
does not affect the schema of the realm. Deleting all objects can be
useful for quickly clearing out your realm while prototyping.

The following code demonstrates how to delete everything
from a realm with [`deleteAll()`](https://docs.mongodb.com/realm-sdks/java/latest/io/realm/Realm.html#deleteAll--):
