## App Services

Realm App Services is a backend for client applications hosted
in the cloud. The Java SDK optionally contains the ability
to access App Services. With App Services, you can:

- Use built-in user management to register
  accounts and [log in](/sdk/java/app-services/users/authenticate-users)
  across devices.

- Back up and synchronize user data across devices with [Sync](/sdk/java/app-services/sync/overview).

- Execute logic on the backend with [Functions](/sdk/java/app-services/functions).

- React to events on the backend with [Triggers](https://www.mongodb.com/docs/realm/triggers/overview/).
