The SDK is the interface to the backend. It provides access to
authentication, Functions, and Sync.

## Prerequisites

You must [initialize the SDK](/sdk/java/quick-start#initialize-realm)
with `Realm.init()` before creating any instance of an `App`.

## Access the App Client

Pass your App ID, which you can [find in the Realm UI](https://www.mongodb.com/docs/realm/get-started/find-your-project-or-app-id/#std-label-find-your-app-id).

:::note Multiple App Instances
You can create multiple `App` instances to connect to multiple
backends or to the same backend with different configurations. All
`App` instances that share the same App ID use the same underlying
connection.
:::
