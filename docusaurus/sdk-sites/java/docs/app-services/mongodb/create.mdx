These code snippets demonstrate how to insert one or more
documents into a MongoDB collection from a mobile application. Insert
operations take a document or documents to add to MongoDB as an argument
and return a [`RealmResultTask`](/sdk/java/async-api/#realmresulttask)
that resolves to an object that contains the results of the operation.

[`collection.insertOne()`](https://www.mongodb.com/docs/realm-sdks/java/latest/io/realm/mongodb/mongo/MongoCollection.html#insertOne-DocumentT-).
