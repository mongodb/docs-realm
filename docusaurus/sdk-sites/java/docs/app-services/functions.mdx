You can use **Functions** to execute logic in the backend. The examples
in this section demonstrate calling a simple function named `sum` that
takes two arguments, adds them, and returns the result:

```javascript
// sum: adds two numbers
exports = function (a, b) {
  return a + b;
};
```

## Call a Function by Name

To execute a function from the SDK, use
[`app.getFunctions()`](https://www.mongodb.com/docs/realm-sdks/java/latest/io/realm/mongodb/App.html#getFunctions-io.realm.mongodb.User-)
to retrieve a [Functions](https://www.mongodb.com/docs/realm-sdks/java/latest/io/realm/mongodb/functions/Functions.html)
manager. Pass the name and parameters of the Function you would like to
call to [`functions.callFunction()`](https://www.mongodb.com/docs/realm-sdks/java/latest/io/realm/mongodb/functions/Functions.html#callFunction-java.lang.String-java.util.List--)
or [`functions.callFunctionAsync()`](https://www.mongodb.com/docs/realm-sdks/java/latest/io/realm/mongodb/functions/Functions.html#callFunctionAsync-java.lang.String-java.util.List--io.realm.mongodb.App.Callback-):
