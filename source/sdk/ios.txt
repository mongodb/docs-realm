=====================
MongoDB Realm iOS SDK
=====================


The {+service+} iOS SDK enables mobile applications to access data
stored in local {+realms+} and interact with {+backend+} services like
Functions, MongoDB Data Access, and authentication. The iOS SDK supports
both Swift and Objective-C apps.
