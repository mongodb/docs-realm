Call a Function by Name
-----------------------

The following example in this section demonstrate calls a simple
:ref:`Realm function <functions>` named ``sum`` that takes two
arguments, adds them, and returns the result:

.. code-block:: javascript

   // sum: adds two numbers
   exports = function(a, b) {
     return a + b;
   };
