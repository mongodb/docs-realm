================================
Update a Collection - Kotlin SDK
================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

To update a collection of objects in a realm:

1. Query a realm for a collection of objects
   with :kotlin-sdk:`realm.query()
   <{+kotlin-local-prefix+}io.realm.kotlin.query/-realm-query/query.html>`.

#. Open a write transaction with :kotlin-sdk:`realm.write()
   <{+kotlin-local-prefix+}io.realm.kotlin/-realm/write.html>` or
   :kotlin-sdk:`realm.writeBlocking()
   <{+kotlin-local-prefix+}io.realm.kotlin/-realm/write-blocking.html>`.

#. Update elements of the set of :kotlin-sdk:`RealmResults
   <{+kotlin-local-prefix+}io.realm.kotlin/-realm-results/index.html>`
   returned by the query.
