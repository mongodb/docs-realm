.. _node-field-types:

========================
Field Types -Node.js SDK
========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol


{+client-database+} supports the following field data types:
- ``bool`` which maps to the JavaScript :mdn:`Boolean </Web/JavaScript/Reference/Global_Objects/Boolean>` type
- ``int`` which maps to the JavaScript :mdn:`Number </Web/JavaScript/Reference/Global_Objects/Number>` type. Internally {+client-database+} stores ``int`` with 64 bits.
- ``float`` which maps to the JavaScript :mdn:`Number </Web/JavaScript/Reference/Global_Objects/Number>` type. Internally {+client-database+} stores ``int`` with 32 bits.
- ``double`` which maps to the JavaScript :mdn:`Number </Web/JavaScript/Reference/Global_Objects/Number>` type. Internally {+client-database+} stores ``int`` with 64 bits.
- ``string`` which maps to the JavaScript :mdn:`String </Web/JavaScript/Reference/Global_Objects/Boolean>` type.
- ``data`` which maps to the JavaScript :mdn:`ArrayBuffer </Web/JavaScript/Reference/Global_Objects/ArrayBuffer>` type.
- ``date`` which maps to the JavaScript :mdn:`Date </Web/JavaScript/Reference/Global_Objects/Date>` type.
