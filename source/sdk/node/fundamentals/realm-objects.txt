=============
Realm Objects
=============

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

The Node.js SDK memory maps objects in a realm directly to JavaScript objects,
called **Realm objects**. For the most part Realm objects work just like common
JavaScript objects, but they also bring along a few additional features.
