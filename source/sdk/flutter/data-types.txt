=========================
Field Types - Flutter SDK
=========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Overview
--------

The Flutter SDK currently only supports Dart-language data types. 
The Flutter SDK **does not** support MongoDB.BSON types or Realm-specific types
at the moment.


.. _flutter-dart-types:

Dart Types
----------

{+client-database+} supports the following Dart language data types:

- ``int``
- ``double``
- ``bool``
- ``String``
- ``null``
- ``List<T>``, where T is any of the supported data types

.. _flutter-embed-realm-object:

Embed RealmObjects
------------------

You can also embed a :flutter-sdk:`RealmObject <realm/RealmObject-class.html>`
in another ``RealmObject`` when creating models.
