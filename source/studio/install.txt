Overview
--------

Realm Studio is available for Linux, MacOS, and Windows. Installing it 
is as easy as selecting a version, and downloading and running the 
installer.

.. tip::
   
   If you want a file type that isn't listed here, such as a ``.zip`` 
   file, view `the release page <https://studio-releases.realm.io/>`_ 
   for the version you want. You'll find a complete list of installers 
   for each version.
