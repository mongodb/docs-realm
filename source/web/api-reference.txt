=======================
API Reference - Web SDK
=======================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

The Web SDK shares API reference documentation with the two other Realm JavaScript SDKs,
the :ref:`Node.Js SDK <node-intro>` and the :ref:`React Native SDK <react-native-intro>`. 

However, the Web SDK doesn't support creating a local Realm database or using :ref:`sync <sync>`. 
Instead, web apps built with the Wek SDK use :ref:`GraphQL <graphql-api>` 
or the :ref:`Query API <web-query-mongodb>` to query data stored in 
{+atlas-short+}. 

Read the :js-sdk:`Realm JavaScript API reference <index.html>`. 

Supported Classes & Namespaces
------------------------------

The reference documentation doesn't differentiate between classes and members
supported by the Web SDK versus those only supported by Node.Js and React Native.

The Realm Web SDK supports the following classes and namespaces 
which are documented in the API reference: 

- :js-sdk:`Realm.BSON <Realm.html#.BSON>`
- :js-sdk:`Realm.App <Realm.App.html>`
- :js-sdk:`Realm.Auth <Realm.Auth.html>`
- :js-sdk:`Realm.Credentials <Realm.Credentials.html>`
- :js-sdk:`Realm.MongoDBCollection <Realm.MongoDBCollection.html>`
- :js-sdk:`Realm.User <Realm.User.html>`
- :js-sdk:`Realm~MongoDB <Realm-MongoDB.html>`
- :js-sdk:`Realm~MongoDBDatabase <Realm-MongoDBDatabase.html>`
