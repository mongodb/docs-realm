.. _app-services-get-started-overview:

The resources on this page are designed to help you begin using Device Sync. 
To explore a working codebase that showcases Device Sync
features, check out a Template App. For a more guided experience in
learning how to develop with Device Sync and Realm, start with a 
Tutorial.

.. _app-services-get-started-template-apps:

Template Apps
-------------

Bootstrap Device Sync with a :ref:`Template App <template-apps>`. 
Template apps bring together many of the building blocks available in 
Atlas App Services and Atlas Device SDK and start you off with a prebuilt 
app that you can customize. The following template apps are available:

- Todo list mobile apps written with Realm SDKs that sync data with App 
  Services using Device Sync
- A todo list web app using the GraphQL API and the Realm Web SDK

To learn more:

- Watch our `Using Template Apps in the Atlas UI
  <https://learn.mongodb.com/learn/course/using-template-apps-in-the-atlas-ui/learning-byte/learn>`__ 
  Learning Byte.

- Check out the template apps in the 
  `Atlas UI <https://www.mongodb.com/realm/register?tck=docs_CTA_realm_template_apps>`__

- Follow along with one of the tutorials below to build a mobile app 
  based on the template app.

.. button:: Sign-up to Deploy a Template App
   :uri: https://www.mongodb.com/realm/register?tck=docs_CTA_realm_template_apps
