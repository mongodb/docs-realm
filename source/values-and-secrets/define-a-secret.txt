===============
Define a Secret
===============

.. default-domain:: mongodb

Overview
--------

You can define a new :ref:`Secret <app-secret>` in the Realm
UI, with the :doc:`Admin API </admin/api/admin-v3>`, or with
the :doc:`Realm CLI </deploy/realm-cli-reference>`. Secrets
are not included in exported application directories.

Procedure
---------

.. include:: /includes/steps/define-a-secret-realm-ui.rst

Summary
-------

- You can only define a Secret using the Realm UI, the :doc:`Admin API </admin/api/admin-v3>`, or the :doc:`Realm CLI </deploy/realm-cli-reference>`.
- Secrets are not exported and cannot be read directly.
