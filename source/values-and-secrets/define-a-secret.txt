===============
Define a Secret
===============

.. default-domain:: mongodb

Overview
--------

You can define a new :ref:`Secret <app-secret>` in the Realm
UI or with the :doc:`Admin API </admin/api/admin-v3>`.
Secrets are not included in
exported application directories and cannot be imported.

---------

.. include:: /includes/steps/define-a-secret-realm-ui.rst

Summary
-------

- You can only define a Secret using the Realm UI or the :doc:`Admin API </admin/api/admin-v3>`.
- Secrets are not exported and cannot be imported.
