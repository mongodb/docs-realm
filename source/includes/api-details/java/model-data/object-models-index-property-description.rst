You can index properties of these data types:

- ``String``
- ``UUID``
- ``ObjectId``
- ``Integer`` or ``int``
- ``Long`` or ``long``
- ``Short`` or ``short``
- ``Byte`` or ``byte[]``
- ``Boolean`` or ``bool``
- ``Date``
- ``RealmAny``

To index a field, use the :java-sdk:`@Index <io/realm/annotations/Index.html>`
annotation.
