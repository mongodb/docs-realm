You can index properties of these data types:

- ``String``
- ``Byte``
- ``Short``
- ``Int``
- ``Long``
- ``Boolean``
- ``RealmInstant``
- ``ObjectId``
- ``RealmUUID``

To create an index on a property, use the :kotlin-sdk:`@Index
<io.realm.kotlin.types.annotations/-index/index.html>` annotation on the
property.
