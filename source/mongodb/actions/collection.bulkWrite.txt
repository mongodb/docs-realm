======================
collection.bulkWrite()
======================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Definition
----------

.. method:: collection.bulkWrite()

Bulk insert/update/delete multiple documents in a collection within a single 
API call and return a ``BulkWriteOpResultObject``.

Usage
-----

Example
~~~~~~~

To call the ``collection.insertOne()`` action from a
:doc:`Function </functions>`, get a collection handle with
:method:`database.collection()` then call the handle's
``insertOne()`` method.

.. include:: /mongodb/crud-snippets/bulkWrite/functions.rst

Parameters
~~~~~~~~~~

The ``collection.bulkWrite()`` action has the following form:

.. code-block:: javascript

   bulkWrite(operations, options, callback)

.. include:: /includes/extracts/mongodb-action-params-bulkWrite.rst

Return Value
~~~~~~~~~~~~

The ``collection.insertOne()`` action returns a :mdn:`Promise
<Web/JavaScript/Reference/Global_Objects/Promise>` that
resolves to a document that describes the insert operation.

.. code-block:: javascript

   Promise<result: document>

.. cssclass:: config-table
.. list-table::
   :header-rows: 1
   :widths: 45 55

   * - Value
     - Description

   * - :guilabel:`Inserted ID`
       
       | ``result.insertedId: <ObjectID>``
     - The ``_id`` value of the document that the insert operation added
       to the collection.
